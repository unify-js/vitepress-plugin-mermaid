# 键盘快捷键

VitePress Mermaid 提供直观的键盘快捷键，用于高效的图表导航。

## 可用快捷键

| 快捷键         | 操作     | 描述               |
| -------------- | -------- | ------------------ |
| `ESC`          | 关闭预览 | 关闭全屏预览模态框 |
| `Ctrl/Cmd + +` | 放大     | 增加缩放级别       |
| `Ctrl/Cmd + -` | 缩小     | 减小缩放级别       |
| `Ctrl/Cmd + 0` | 重置缩放 | 重置为原始缩放级别 |

## 鼠标交互

| 操作            | 功能           |
| --------------- | -------------- |
| **点击** 图表   | 打开全屏预览   |
| **滚轮滚动**    | 在光标位置缩放 |
| **拖拽**        | 放大时平移     |
| **点击 X 按钮** | 关闭预览       |
| **点击背景**    | 关闭预览       |

## 触摸手势（移动端）

| 手势            | 功能         |
| --------------- | ------------ |
| **点击** 图表   | 打开全屏预览 |
| **捏合**        | 放大/缩小    |
| **滑动**        | 放大时平移   |
| **点击 X 按钮** | 关闭预览     |

## 使用示例

### 快速导航

1. 点击图表打开
2. 多次按 `Ctrl + +` 放大
3. 拖拽到感兴趣的区域
4. 按 `ESC` 完成并关闭

## 缩放级别

预览支持广泛的缩放级别：

- **最小**：0.1x（原始大小的 10%）
- **最大**：10x（原始大小的 1000%）
- **步进**：0.1x 增量

## 无障碍访问

所有键盘快捷键都设计为：

- **可发现** - 在界面中显示
- **一致** - 匹配常见模式（如浏览器缩放）
- **不干扰** - 不影响页面导航

### 焦点管理

当预览打开时：

- 焦点移动到模态框容器
- `Tab` 导航被限制在模态框内
- 按 `ESC` 将焦点返回到触发元素

## 自定义

目前，键盘快捷键是固定的以保持一致性。未来版本可能允许通过配置进行自定义。

## 技巧

1. **先缩放再平移** - 先放大，然后拖拽探索
2. **使用键盘进行精确控制** - 键盘快捷键比滚动更精细
3. **迷路时重置** - `Ctrl/Cmd + 0` 快速返回完整视图
4. **ESC 是好帮手** - 关闭并继续阅读的最快方式
